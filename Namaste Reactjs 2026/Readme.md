# 🚀 Namaste React - From Zero to Hero

Welcome to my learning repository for the **[Namaste React](https://namastedev.com/learn/namaste-react)** course. This repository contains all the code implementations, concepts, and major projects built during this journey to master React.js, moving from foundational principles to advanced, production-ready architecture.

## 🎯 Course Overview

This curriculum strictly focuses on **project-based learning** and **industry-grade best practices**, covering everything from raw HTML/JS setups to advanced React hooks, state management, optimization techniques, and testing.

### 🛠️ Tech Stack & Tools
* **Core:** React 18, JavaScript (ES6+), HTML5, CSS3
* **State Management:** Redux Toolkit, Context API
* **Styling:** Tailwind CSS
* **Routing:** React Router DOM
* **Bundler:** Parcel
* **Testing:** Jest, React Testing Library
* **Other:** Firebase (Authentication), OpenAI API (GPT-3.5)

## 📚 Course Curriculum (Episode Breakdown)

* **Episode 01 | Inception:** Visual Studio Code setup, understanding "Hello World" in raw HTML/JS vs. React, and injecting React into a project via CDN.
* **Episode 02 | Igniting our App:** Package managers (`npm`), setting up module bundlers (Parcel), understanding `node_modules`, and standardizing the React file structure.
* **Episode 03 | Laying the Foundation:** Deep dive into JSX, understanding React Components, Component Composition, and comparing Functional vs. Class-based syntax.
* **Episode 04 | Talk is Cheap, Show Me the Code:** Building the core React app UI, passing data with `props`, understanding local `state`, and rendering lists using `.map()`.
* **Episode 05 | Let's Get Hooked:** Standardizing file structures, modular Import/Export patterns, and an introduction to the power of React Hooks (specifically `useState`).
* **Episode 06 | Exploring the World:** Monolith vs. Microservice Architecture, utilizing the `useEffect` hook, fetching live API data (Swiggy API), and implementing Conditional Rendering.
* **Episode 07 | Finding the Path:** Mastering `react-router-dom` for Single Page Applications (SPAs), dynamic routing, handling route errors, and utilizing the `<Outlet />` component.
* **Episode 08 | Let's Get Classy:** Unpacking Class-based components and React lifecycle methods (`componentDidMount`, `componentDidUpdate`, `componentWillUnmount`).
* **Episode 09 | Optimizing our App:** Building Custom Hooks to abstract logic, Performance Optimization via Code Splitting / Chunking, Lazy Loading, and the `<Suspense />` component.
* **Episode 10 | Jo Dikhta Hai, Vo Bikta Hai:** Integrating Tailwind CSS and configuring it to build highly responsive, modern UIs efficiently.
* **Episode 11 | Data is the New Oil:** Managing data flow, handling Prop Drilling, utilizing the React Context API, Higher-Order Components, and Controlled vs. Uncontrolled Components.
* **Episode 12 | Let's Build Our Store:** Setting up Redux Toolkit (RTK), configuring the Redux store, creating slices, writing reducers, and dispatching actions.
* **Episode 13 | Time for Test:** Configuring Jest and React Testing Library, understanding unit testing principles, and writing robust test cases for React components.
* **Episode 14–16 | NetflixGPT Series:** End-to-end development of a Netflix clone, integrating Firebase Authentication, and hooking up the OpenAI GPT-3.5 API for an intelligent movie search experience.
* **Bonus Series:** 
  * Machine coding the *Namaste YouTube* clone (Live Chat, Debouncing, Nested Comments).
  * Building the *DevTinder* UI architecture.
  * Deep dive into advanced hooks: `useMemo`, `useCallback`, and `useRef`.

## 🏗️ Major Projects

### 1. 🍔 Namaste Food (Swiggy Clone)
A full-fledged food delivery web application fetching live data from a Swiggy-like API.
* **Key Features:** Live API data fetching, Shimmer UI for loading states, Search & Filter functionality, and a dynamic Cart.
* **Concepts Applied:** Custom Hooks, Higher-Order Components, Routing, Redux Toolkit, and Tailwind CSS.

### 2. 🎬 NetflixGPT
A movie recommendation application integrating OpenAI's GPT-3.5 Turbo for intelligent search.
* **Key Features:** User Authentication (Sign In/Up/Out), Multi-language support, GPT-powered movie search, and secure API key management using `.env`.
* **Concepts Applied:** Form Validation, `useRef`, Memoization (`useMemo`, `useCallback`), and Firebase Authentication.

### 3. 📺 Namaste YouTube
A highly optimized YouTube clone handling real-time data and advanced UI interactions.
* **Key Features:** Live YouTube search with autocomplete, Nested Comments (N-level deep), Live Chat system (API Polling), and Infinite Scroll.
* **Concepts Applied:** API Caching, Debouncing, Machine Coding principles, and advanced Redux state management.

### 4. 💻 DevTinder (UI)
The frontend architecture for a developer-focused social connection platform.
* **Key Features:** Dynamic routing, user interaction flows, and real-time state updates.
